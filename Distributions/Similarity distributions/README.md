### Format:
The similarity matrix $S \in {\mathbb{R}^{{N_{train}} \times {N_{test}}}}$.
|       | $test_{0}$ | $test_{1}$ | ... | $test_{N_{test}}$ |
| :-----| ----: | :----: | :----: | :----: |
| $train_{0}$ | 0.5 |0.3  |0.3  |0.3  |
| $train_{1}$ | 1 | 0.2 |0.3  |0.3  |
| ... | 1 | 0.2 |0.3  |0.3  |
| $train_{N_{train}}$ | 1 | 0.2 |0.3  |0.3  |
